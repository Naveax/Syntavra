# Roblox Profile Benchmark Results

The committed controlled result is generated by:

```bash
python benchmarks/roblox_profile_benchmark.py --cases 50 --mode simulated --output benchmarks/results/roblox-profile/simulated-50.json
```

Only the `simulated` mode has a committed result. Transcript and live modes intentionally return unavailable/partial metadata until sanitized transcript artifacts or controlled live runs exist.

The benchmark does not compare SignalCore with third-party products. Every competitor group is reported as `NOT_COMPARABLE` until identical-model, identical-provider, identical-task, identical-permission, and identical-verifier execution is available.
